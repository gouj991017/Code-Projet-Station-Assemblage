TITLE: Code de génération d'instructions pour le programme "Terminal_Base_V2.0".
DESCRIPTION: S'occupe des fonctions expliquées dans le «README.TXT» précédant.
Il s'agit d'un programme permettant de créer, modifier, ouvrir et sauvegarder des fichiers d'instructions pour execution dans
le programme "Terminal_Base_V2.0". Afin de créer un fichier d'instructions, il suffit de créer autant d'étapes et de pièces
que désiré et de les sauvegarder via le menu.
Le code a été programmé en Java.

Emplacement et nom du fichier contenant le corps du code:
-Générateur d'instructions\src\générateur\d\instructions\UI_Generateur.java
Nom des classes nécéssaires (dans Générateur d'instructions\src\générateur\d\instructions)
-Etape.java
-Piece.java

Le logiciel NetBeans:
-Permet la création d'interface.
-Possède un intellisense de base permettant de repérer les erreurs plus facilement.
-Pour compiler, il suffit de sauvegarder le code ou d'exécuter le programme en mode de dépannage (dans les options du menus en haut de la fenêtre).
-Le programme s'exécute normalement en appuyant sur la flèche verte près de l'option de dépannage dans les options du menus en haut de la fenêtre.

MATERIAL NEEDED:
-Tout ordinateur possédant le JDK 14 de java.
